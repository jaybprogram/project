For Lab 7, we were instructed to create a table that resembles a tier list. You might be wondering, what is a table? A table in HTML is a way to organize information into rows and columns, similar to a chart or grid. It’s useful for presenting data clearly, such as schedules, lists, or comparisons. To create a table, you start with table, use tr for rows, th for headers, and td for the data within the rows. Click on the image to get a better look into the project!

Lab 09 XML

For lab 9 we were instructed to fix and organize employee data. XML is like a way to organize data using tags. It’s similar to HTML but is used to store and transport data, not just display it. Each piece of data is wrapped in custom tags that you create, so computers can understand and share the information easily.
